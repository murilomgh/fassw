Ferramenta de Anotação Semântica de Serviços Web

Trata-se de um protótipo desenvolvido em Java cuja finalidade é gerar descrições semânticas em WSMO (Web Services Modeling Ontology) à partir de descrições de serviços Web escritas WSDL (Web Services Description Language).

O programa recebe como entrada arquivos em WSDL 2.0* e fornece como saída um ou mais de arquivos WSMO Ontology e um WSMO Web Service, escritos em WSML (Web Services Modeling Language).

O processo de conversão estabelece um mecanismo simples de grounding em WSMO. A ferramenta utiliza a biblioteca Apache Woden para o parsing dos documentos WSDL.

* Arquivos WSDL 1.1 podem, opcionalmente, ser convertidos para WSDL 2.0
